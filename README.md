## 👨‍💻 About Me

<img align="right" src="https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExM2FqZ2Z5YmYzY3Z1cGFxdHByNnp5ZWwxaHExdnE3eW80cDFwNnEzeiZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/L1R1tvI9svkIWwpVYr/giphy.gif" width="250" alt="Coding GIF" />

I'm **Sebastián**, a Software Engineering student at **UPC** (Universidad Peruana de Ciencias Aplicadas) based in Lima, Peru. 

My trajectory is defined by a deep curiosity for how software interacts with the physical world, pushing me into the realms of hardware integration, autonomous vehicles, and algorithmic optimization. When I'm not debugging memory leaks in C++, I'm probably organizing events, reading philosophy, or training in the pool.

- ⚙️ Building **Embedded Systems** & **IoT** solutions from the ground up
- 🤖 Optimizing paths for **Autonomous Underwater Vehicles (AUVs)** using graph algorithms
- 📋 Serving as **Logistics Coordinator** for the **IEEE Computer Society** at UPC
- 🏊‍♂️ **Competitive Swimmer** & Certified Instructor (FDPN)
- 🎙️ Hosted academic events like *La Noche de la Filosofía*
